# Disease Prediction from Medical Data

## Overview

This project predicts whether a patient has malignant or benign breast cancer using machine learning techniques. The model was trained and evaluated on the Breast Cancer Wisconsin Dataset.

## Objective

To build a disease prediction model that classifies medical data accurately using machine learning algorithms.

## Dataset

* Breast Cancer Wisconsin Dataset
* Total Samples: 569
* Features: 30

## Algorithms Used

* Logistic Regression
* Random Forest Classifier

## Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Results

* Logistic Regression Accuracy: 97%
* Random Forest Accuracy: 96%

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix

## Conclusion

Logistic Regression performed slightly better than Random Forest for this dataset, achieving 97% accuracy and demonstrating effective disease prediction using machine learning.
